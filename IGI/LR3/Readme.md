## Задания (Вариант 25)

1. **Задание 1**: Вычислить \( e^x \) с помощью разложения в степенной ряд с точностью `eps`. Вывести результат в виде таблицы: `x`, `F(x)`, `n`, `Math F(x)`, `eps`. Использован декоратор для измерения времени выполнения.
2. **Задание 2**: Подсчитать сумму каждого второго введённого целого числа до ввода числа `1`.
3. **Задание 3**: Подсчитать количество слов, начинающихся со строчной согласной буквы, в строке, введённой пользователем.
4. **Задание 4**: Проанализировать заданную строку:
   - Подсчитать слова минимальной длины.
   - Вывести слова, за которыми следует точка.
   - Найти самое длинное слово, заканчивающееся на 'r'.
5. **Задание 5**: Обработать список вещественных чисел:
   - Найти минимальный элемент по модулю.
   - Найти сумму элементов между первым и последним положительными элементами.
   - Список можно ввести вручную или сгенерировать с помощью генератора.

## Структура проекта

- `main.py`: Основной скрипт для запуска всех заданий.
- `series_expansion.py`: Логика для Задания 1 (вычисление \( e^x \)).
- `initialization.py`: Функции для инициализации последовательностей (ввод и генератор).
- `text_analysis.py`: Функции для анализа строк (Задания 3 и 4).
- `list_processing.py`: Обработка списков (Задание 5).
